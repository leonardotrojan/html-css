Quebra de linha, linhas

para adicionar uma linha basta usar <hr>

para quebrar a linha, apenas usar <br>

Emojis, imagens e favicons

para adicionar um emoji basta colocar "& + código do emoji da emojipedia

Para adicionar uma imagem basta utilizar o comando <img e selecionar a imagem que deve estar na mesma pasta

para adicionar um favicon basta ir na seção <head> e digitar a tag <link e selecionar o icone que deve estar na mesma pasta

hierarquia de títulos

a hierarquia vai desde o <h1> até o <h6>

formatações

existem várias, como <strong> (para negrito); <EM> (para itálico); <big> e <small> para grande e pequeno respectivamente; o <mark para marcadores; <del> e <ins> para texto deletado e inserido respectivamente; e <sup> e <sub> para texto sobrescrito e texto subscrito respectivamente; <code> para as letras ficarem com o mesmo espaçamento; <pre> para que seja adotado a pré-formatação da escrita do código;
